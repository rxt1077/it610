Systems Administration Resources

This website houses presentations and exercises for teaching Linux systems administration with containers.

All resources are created using Typst, are available on GitHub, and are licensed under CC BY-NC 4.0

The current syllabus can be found here.

Italics indicated that a presentation or exercise still needs to be converted to use the Typst template.

Exercises

The exercises are listed in the order they are used with the learning goals that they cover.

  1. Getting Started

    • Install git
    • Install Docker
    • git pull the class git repository
    • Build a container image
    • Run a container

  2. Creating a Container Image

    • Use users and permissions in Linux
    • Create a custom container image
    • Build a container image
    • Run a container

  3. Exercise 3

    • Installing GAM on an Ubuntu container via their installation script
    • Creating a Debian package for GAM
    • Creating a Dockerfile for a container that installs GAM from your package

  4. Volumes

    • Create a new volume
    • Run a container with a volume mounted
    • Create data on a volume
    • Back up a volume

  5. Hello Kubernetes

    • Start the built-in version of Kubernetes in Docker Desktop
    • Build a custom image for a web app
    • Create a Kubernetes Deployment
    • Create a Kubernetes Service

  6. Exploring a Kubernetes Database Deployment

    • Apply Kubernetes objects from a YAML file
    • Use kubectl to explore a running system

  7. Docker Compose

    • Run a Docker Compose system
    • Examine a functioning Docker Compose system

  8. Creating a Developer Environment

    • Create a multi-container development environment using a container orchestration tool
